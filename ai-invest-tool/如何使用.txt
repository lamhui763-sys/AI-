✅ 解決方法（超簡單）
================================

第1步：打開這個文件夾
------------------------
在文件管理器地址欄輸入：
C:\Users\makai\WorkBuddy\Claw\ai-invest-tool

按Enter


第2步：雙擊這個文件
------------------------
找到並雙擊：
RUN_ME.bat


第3步：等待啟動
------------------------
- 如果看到「找不到Python」：
  按照提示安裝Python
  
- 如果看到「正在安裝依賴」：
  等待2-5分鐘

- 如果看到「正在啟動Web服務器」：
  等待幾秒鐘

- 瀏覽器會自動打開 http://localhost:8501


================================
⚠️ 重要提示
================================

啟動後：

1. 不要關閉黑色的命令窗口！
   - 這個窗口必須保持打開
   - 關閉它會停止應用

2. 如果瀏覽器沒有自動打開
   - 手動打開瀏覽器
   - 輸入：http://localhost:8501

3. 使用完畢後
   - 在命令窗口按 Ctrl+C
   - 或直接關閉命令窗口


================================
🔍 如果RUN_ME.bat無法運行
================================

情況1：雙擊後閃退
--------------------
解決：
1. 可能Python未安裝
2. 安裝Python：https://www.python.org/downloads/
3. 安裝時勾選「Add Python to PATH」
4. 重啟電腦
5. 再次運行RUN_ME.bat


情況2：顯示「找不到Python」
---------------------------
解決：
1. 下載Python：https://www.python.org/downloads/
2. 運行安裝程序
3. ⚠️ 勾選「Add Python to PATH」
4. ⚠️ 勾選「Install for all users」
5. 完成安裝
6. 重啟電腦
7. 再次運行RUN_ME.bat


情況3：安裝依賴失敗
--------------------
解決：
1. 檢查網絡連接
2. 以管理員身份運行RUN_ME.bat
   （右鍵 -> 以管理員身份運行）
3. 或手動安裝：
   pip install streamlit yfinance pandas plotly openpyxl
