## 💡 快速使用示例

### 示例1：分析恒生指數

1. 點擊"股票分析"頁面
2. 在股票代碼框中輸入：`^HSI`
3. 選擇時間周期：1年
4. 點擊"開始分析"
5. 查看結果：
   - 📈 價格圖表
   - 📊 技術指標
   - 🎯 交易信號
   - 📝 AI建議
6. 點擊"導出為Excel"保存報告

### 示例2：查看技術指標

1. 切換到"技術分析"頁面
2. 輸入股票代碼：`6158.HK`（海爾智家）
3. 勾選指標：RSI、MACD、布林帶
4. 點擊"生成分析"
5. 與圖表交互（縮放、平移）
6. 查看詳細指標數據

### 示例3：使用AI分析

1. 切換到"AI分析"頁面
2. 輸入股票代碼：`7200.HK`
3. （可選）輸入您的OpenAI API Key
4. 選擇分析類型：基礎分析
5. 點擊"開始AI分析"
6. 閱讀AI的投資建議
