# 📸 界面预览和功能演示

## 🎯 快速开始

**最简单的方法** - 双击运行 `start.bat`（Windows）或运行 `python run.py`

浏览器会自动打开 **http://localhost:8501**

### 1. 📊 股票分析页面

**功能说明**:
- 输入股票代码并分析
- 查看实时价格走势
- 获取交易信号
- 查看技术指标

**使用步骤**:
1. 在侧边栏输入股票代码（如 `^HSI`）
2. 选择时间周期（1月、3月、6月、1年、2年、5年）

**展示内容**:

## 🎯 使用技巧

### 技巧1: 快速切换股票

在侧边栏输入股票代码后，所有页面都会使用这个股票进行分析。

### 技巧2: 导出Excel报告

在任何分析页面，点击"导出为Excel"按钮，即可生成专业的Excel报告。

### 技巧3: 多股票对比

在"股票分析"页面，输入多个股票代码（用逗号分隔），可以同时分析多只股票。

### 技巧4: 参数调优

在"技术分析"和"回测引擎"页面，可以实时调整参数，图表会自动更新。

### 技巧5: 实时更新

修改代码后，Web界面会自动刷新，无需手动重启。

## 📊 典型工作流

### 每日分析流程

1. **股票分析** → 查看关注列表的交易信号
2. **技术分析** → 深入分析感兴趣的技术指标
3. **AI分析** → 获取智能投资建议
4. **投资组合** → 更新持仓，查看盈亏

### 策略研发流程

1. **技术分析** → 发现潜在的交易信号
2. **回测引擎** → 验证策略有效性
3. **AI分析** → 获取AI对策略的看法
4. **导出报告** → 生成详细分析报告
