⚠️ 如果看到「找不到Python」

2. 下載Python 3.8或更高版本

3. 運行安裝程序

4. ⚠️ 重要：安裝時務必勾選：
   [√] Add Python to PATH

5. 安裝完成後，重新雙擊 START_HERE.bat
